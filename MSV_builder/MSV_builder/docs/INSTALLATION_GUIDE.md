## 🔧 УСТАНОВКА

### Шаг 1: Найдите папку плагинов SketchUp

**Windows:**
```
C:\Users\[ваше_имя]\AppData\Roaming\SketchUp\SketchUp 2024\SketchUp\Plugins\
```

Быстрый способ:
1. В SketchUp: Window → Preferences → Extensions
2. Нажмите "Install Extension..."
3. Путь откроется в проводнике

**Или через Ruby Console:**
```ruby
# В SketchUp: Window → Ruby Console
puts Sketchup.find_support_file "Plugins"
```

### Шаг 2: Найдите папку MSV_builder

В папке Plugins найдите:
```
Plugins\
  MSV_builder\
    MSV_MSV_MSV_builder_enhanced.rb  ← старый файл (или другой .rb файл)
    bed_settings.json        ← настройки (можно оставить)
    (возможны другие файлы)
```

### Шаг 3: Создайте резервную копию

**ОБЯЗАТЕЛЬНО!** Скопируйте старый файл:
```
MSV_MSV_MSV_builder_enhanced.rb → MSV_MSV_builder_enhanced_BACKUP.rb
```

### Шаг 4: Замените файл

1. Удалите старый `MSV_MSV_MSV_builder_enhanced.rb`
2. Скопируйте новый `MSV_MSV_MSV_builder_enhanced.rb` в папку `MSV_builder`

### Шаг 5: Проверьте loader файл

Найдите файл загрузчика в папке Plugins:
```
Plugins\
  MSV_builder_loader.rb  ← должен быть здесь
```

Откройте его и проверьте, что он загружает правильный файл:
```ruby
# Должно быть примерно так:
module BedBuilder
  # ...
  extension.creator     = "..."
  extension.copyright   = "..."
  extension.description = "..."
  
  # ВАЖНО: Проверьте эту строку
  Sketchup.register_extension(extension, true)
end
```

Если loader файла нет, создайте его:

```ruby
# MSV_builder_loader.rb
require 'sketchup.rb'
require 'extensions.rb'

module BedBuilder
  unless file_loaded?(__FILE__)
    ex = SketchupExtension.new('MSV Bed Builder', 'MSV_builder/MSV_MSV_builder_enhanced')
    ex.description = 'Конструктор кровати из МДФ с валидацией и предпросмотром'
    ex.version     = '2.0.1'
    ex.copyright   = '© 2026'
    ex.creator     = 'MSV'
    Sketchup.register_extension(ex, true)
    file_loaded(__FILE__)
  end
end
```

### Шаг 6: Перезапустите SketchUp

**Полностью закройте** SketchUp и запустите снова.

Window → Ruby Console

1. Нажмите **Plugins → Bed Builder → Построить кровать**
2. Должен открыться диалог с:
   - Формой параметров слева

6. Кровать должна появиться в сцене

## ❌ ЕСЛИ ЕСТЬ ОШИБКИ

### Ошибка: "unexpected end"
**Причина:** Файл поврежден при копировании  
**Решение:** Скачайте файл заново и убедитесь, что копируете полностью

### Ошибка: "cannot load such file"
**Причина:** Неправильный путь в loader файле  
**Решение:** Проверьте, что в loader файле указан правильный путь:
```ruby
'MSV_builder/MSV_MSV_MSV_builder_enhanced.rb'
```

### Ошибка: Плагин не появляется в меню
**Причина:** Плагин не загружен  
**Решение:** 
1. Проверьте наличие loader файла
2. Проверьте консоль на ошибки
3. Попробуйте загрузить вручную:
```ruby
# В Ruby Console
load 'C:/Users/user/AppData/Roaming/SketchUp/SketchUp 2024/SketchUp/Plugins/MSV_builder/MSV_MSV_MSV_builder_enhanced.rb'
```

### Ошибка: Диалог не открывается
**Причина:** Ошибка в HTML коде  
**Решение:**
1. Проверьте консоль на ошибки JavaScript
2. Убедитесь, что файл скопирован полностью

## 🎯 ПЕРВЫЙ ЗАПУСК

### Создайте тестовую кровать:

1. **Plugins → Bed Builder → Построить кровать**

2. **Заполните параметры:**
   ```
   ОСНОВНЫЕ ПАРАМЕТРЫ:
   - Ширина матраса: 1600 мм
   - Длина матраса: 2000 мм
   - Отступ от матраса: 10 мм
   - Высота царг: 250 мм
   - Толщина МДФ: 16 мм
   - Высота ножек: 150 мм
   - Утопление матраса: 50 мм
   
   КОНСТРУКЦИЯ ЦАРГ:
   - Тип основания: Центральная царга
   - Ориентация: Поперек
   - Высота центр. царги: До пола
   - Двойные царги: Нет
   - Задняя царга: До пола
   
   ИЗГОЛОВЬЕ:
   - Строить спинку: Да
   - Высота спинки: 900 мм
   - Толщина спинки: 16 мм
   - Выступы спинки: 50 мм
   
   (остальное оставить по умолчанию)
   ```

3. **Проверьте предпросмотры:**
   - Вид сверху: матрас, царги, спинка
   - Вид сбоку: ножки, царги, центральная царга до пола, спинка от пола

4. **Нажмите "Создать"**

5. **Проверьте результат:**
   - Кровать появилась в сцене
   - Спинка начинается от пола
   - Центральная царга идет от пола
   - Сообщение: "Кровать создана успешно!"
