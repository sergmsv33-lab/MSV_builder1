# MSV Bed Builder v2.0.3

## Исправления

- HTML-интерфейс вынесен из Ruby-кода в отдельный файл `MSV_builder/ui/index.html`.
- Исправлено построение изголовья и боковой спинки: спинки теперь строятся от пола.
- Исправлен боковой предпросмотр: добавлены отображение царг, двойных царг, центральной царги, дна, спинок и фурнитуры.
- В третьем блоке предпросмотра убраны дублирующие габариты, которые уже показаны на видах сверху и сбоку.
- Двойные царги теперь создают не только переднюю и заднюю, но и внутренние боковые царги.
- Добавлены опорные уголки на внутренних гранях царг для опоры матраса.
- Добавлены стяжные угловые уголки для соединения боковых царг с передней и задней царгами.
- В разделе «Конструкция царг» поле «Высота центр. царги» переименовано в «Низ центр. царги».
- Добавлено новое поле «Высота центр. царги (мм)».
- В разделе «Боковая спинка» значение по умолчанию установлено «Нет».
- В разделе «Фурнитура» переименованы поля:
  - «Автоматические фиксаторы» → «Стяжка уголки»
  - «Автоматические стопоры» → «Опорные уголки»
