# 🚀 БЫСТРЫЙ СТАРТ - BED BUILDER v2.0

## ЧТО НОВОГО?

### 🎯 Главные улучшения:
1. **Высота центральной царги** - теперь можно задать отдельно от боковых царг
2. **Валидация** - плагин проверяет корректность размеров ДО создания
3. **Предпросмотр** - видите результат в реальном времени
4. **Карта раскроя** - автоматический расчет всех деталей с экспортом в CSV
5. **Память настроек** - последние параметры сохраняются автоматически
6. **Компоненты уголков** - автоматическая загрузка готовых моделей

## ⚙️ НОВЫЙ ПАРАМЕТР: ВЫСОТА ЦЕНТРАЛЬНОЙ ЦАРГИ

### Зачем нужен?
Позволяет сделать центральную царгу:
- **Ниже** боковых царг (экономия материала)
- **Выше** боковых царг (усиленная конструкция)
- **Равной** боковым царгам (стандарт)

### Пример использования:

#### Стандартная кровать:
```
Высота боковых царг: 250 мм
Высота центр. царги: 250 мм
Результат: все царги одинаковые
```

#### Экономная версия:
```
Высота боковых царг: 250 мм
Высота центр. царги: 150 мм
Высота центр. царги: "В уровень боковых царг"
Результат: центральная царга ниже, экономия МДФ ~40%
```

#### Усиленная конструкция:
```
Высота боковых царг: 250 мм
Высота центр. царги: 300 мм
Высота центр. царги: "До пола"
Результат: центральная царга до пола, максимальная жесткость
```

### ⚠️ Важно:
Если выбрано **"В уровень боковых царг"**, то:
- Высота центр. царги ≤ Высоты боковых царг
- Иначе будет ошибка валидации

## ✅ ВАЛИДАЦИЯ

### Что проверяется:

#### Минимальные значения:
- ✓ Ширина матраса: ≥ 500 мм
- ✓ Длина матраса: ≥ 1000 мм
- ✓ Толщина МДФ: ≥ 6 мм
- ✓ Высота царг: ≥ 100 мм

#### Положительные значения:
- ✓ Все размеры > 0
- ✓ Проверяются: mw, ml, th, tt, lh, hh, ht, cch, и др.

#### Геометрическая совместимость:
- ✓ Матрас помещается в каркас
- ✓ Центральная царга не выше боковых (если выбрано)
- ✓ Утопление матраса < высоты царг

### Как выглядит ошибка:
```
⚠ Ошибки валидации:
• Ширина матраса минимум 500мм
• Высота центр. царги (300мм) > высоты боковых (250мм)
• Утопление матраса должно быть < высоты царг
```

### Что делать при ошибке:
1. Прочитайте список ошибок (красный блок вверху)
2. Исправьте проблемные значения
3. Ошибки исчезнут автоматически
4. Кнопка "Создать" станет активной

## 🔧 КОМПОНЕНТЫ УГОЛКОВ

### Что это?
Вместо простой геометрии - использование готовых 3D моделей уголков

### Как подключить:

#### Шаг 1: Создайте компонент уголка
```
1. В SketchUp создайте модель уголка
2. Сделайте его компонентом
3. Установите точку вставки (Origin) в нужное место
4. Сохраните как: support_bracket_component.skp
```

#### Шаг 2: Поместите в папку плагина
```
Plugins/
  MSV_builder.rb
  support_bracket_component.skp  ← сюда
```

#### Шаг 3: Перезагрузите плагин
```
Plugins → Bed Builder → Перезагрузить плагин
```

### Автоматическое размещение:
При создании кровати с центральной царгой:
- ✅ Компонент загружается автоматически
- ✅ Экземпляры размещаются в нужных точках
- ✅ Автоповорот в зависимости от ориентации царги

### Если компонента нет:
- ⚠️ Предупреждение в консоли: "Support bracket component not found"
- ✅ Создаются простые геометрические уголки (fallback)
- ✅ Функциональность не нарушается

## ❓ ЧАСТЫЕ ВОПРОСЫ

### В: Как изменить кровать после создания?
**О:** Выберите группу → Plugins → Bed Builder → Изменить размеры

### В: Можно ли получить только карту раскроя без 3D?
**О:** Да! Просто нажмите "📋 Карта раскроя" вместо "Создать"

### В: Ошибка "Высота центр. царги больше боковых"?
**О:** Измените опцию на "До пола" или уменьшите высоту центр. царги

### В: Как вернуть значения по умолчанию?
**О:** Удалите файл `bed_settings.json` из папки плагина

### В: Уголки не загружаются?
**О:** Проверьте наличие файла `support_bracket_component.skp`. Если его нет - создадутся простые геометрические уголки

### В: Как посчитать стоимость?
**О:** В CSV файле будет секция СТОИМОСТЬ (если задан параметр mdf_price)

### В: Старые кровати можно редактировать?
**О:** Да, плагин автоматически мигрирует старые параметры

### В: Предпросмотр не обновляется?
**О:** Проверьте консоль на ошибки JavaScript (F12 в некоторых версиях SketchUp)

## 🐛 УСТРАНЕНИЕ ПРОБЛЕМ

### Проблема: Плагин не загружается
```
Решение:
1. Window → Ruby Console
2. Проверьте ошибки
3. Убедитесь, что файл в правильной папке Plugins
4. Перезапустите SketchUp
```

### Проблема: Кнопка "Создать" не активна
```
Причина: Ошибки валидации
Решение:
1. Прочитайте красный блок вверху
2. Исправьте указанные проблемы
3. Кнопка станет активной автоматически
```

### Проблема: CSV с кракозябрами в Excel
```
Причина: Кодировка UTF-8
Решение (Windows):
1. Откройте CSV в Notepad++
2. Encoding → UTF-8 with BOM
3. Сохраните
4. Откройте в Excel
```

### Проблема: Предпросмотр пустой
```
Решение:
1. Заполните основные параметры
2. Убедитесь, что нет ошибок валидации
3. Обновите браузер (если HtmlDialog)
```
