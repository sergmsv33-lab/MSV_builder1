# MSV Bed Builder v2.0.5

## Изменения

- Обновлены значения по умолчанию:
  - ориентация центральной царги: `Вдоль`;
  - низ центральной царги: `В уровень боковых царг`;
  - высота центральной царги: `250 мм`;
  - двойные царги: `Да`;
  - наличие дна: `Да`.
- `Опорные уголки` переименованы в `Опорные бруски матраса` и перенесены в раздел `Дно и Опоры`.
- Для опорных брусков матраса добавлены параметры: длина, высота, толщина.
- В HTML визуально разделены `Бруски для дна` и `Опорные бруски матраса`.
- В карту раскроя добавлен расчет погонных метров отдельно для брусков дна и брусков матраса.
- Геометрические элементы кровати создаются как компоненты, получают имя, атрибуты и метки для удобной работы с OpenCutList.
- Верхний предпросмотр развернут спинкой вверх.
- Боковой предпросмотр развернут спинкой влево.
- На боковом предпросмотре убрана задняя ножка у спинки.
- Блок `Параметры конструкции` сделан компактнее: уменьшены шрифт и межстрочный интервал.

В ZIP включены только измененные файлы версии.
